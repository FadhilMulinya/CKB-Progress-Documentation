Get Started with the Demo

Follow these steps to build and load the extension locally on your machine.

  • 01
    Clone the Repository git clone git@github.com:FadhilMulinya/CKB-Progress-Documentation.git
  • 02
    Navigate & Build cd Month-2/week-4 && pnpm install && pnpm build
  • 03
    Load in Chrome

    Go to chrome://extensions, enable Developer Mode, and click Load Unpacked. Select the dist folder.
